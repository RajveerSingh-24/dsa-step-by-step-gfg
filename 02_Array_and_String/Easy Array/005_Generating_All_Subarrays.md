# Generating All Subarrays

## Problem

Given an array of `N` elements, generate and print all possible **contiguous subarrays**.

A subarray is a continuous part of an array.

## Approach

- Use three nested loops:
  - The first loop selects the starting index.
  - The second loop selects the ending index.
  - The third loop prints all elements between the start and end indices.

## Code (Java)

```java
class Solution {
    static void generateSubarrays(int[] arr) {
        int n = arr.length;

        for (int start = 0; start < n; start++) {
            for (int end = start; end < n; end++) {

                for (int i = start; i <= end; i++) {
                    System.out.print(arr[i] + " ");
                }

                System.out.println();
            }
        }
    }
}
```

## Example

Input:

```text
arr = [1, 2, 3]
```

Output:

```text
1
1 2
1 2 3
2
2 3
3
```

## Explanation

```text
All contiguous subarrays are generated by
choosing every possible start and end index.
```

## Complexity Analysis

- **Time Complexity:** O(N³)
- **Space Complexity:** O(1)
